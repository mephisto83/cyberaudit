# CyberAudit

AI-powered security scanning and vulnerability detection.

## Features
- NIST Cybersecurity Framework compliance scoring
- Automated vulnerability detection
- Risk prioritization with AI-powered severity assessment
- Continuous monitoring with alerting
- React + Tailwind SaaS webapp with Firebase Auth

## Quick Start
1. Clone this repo
2. Install: `pip install -e .`
3. Run: `cyberaudit --help`

## Web App
The `webapp/` directory contains a full SaaS frontend built with React + Tailwind CSS.
